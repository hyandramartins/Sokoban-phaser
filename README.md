# Sokoban Phaser

Este projeto é uma implementação do clássico jogo **Sokoban** utilizando o framework **Phaser** e o build tool **Vite**.

## 🛠️ Pré-requisitos

Para rodar este projeto, é necessário ter instalado em sua máquina:
* **Node.js**
* **JavaScript** e **TypeScript** configurados.

## Instalação e Execução

Siga os passos abaixo no seu terminal para configurar o ambiente local:

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/hyandramartins/Sokoban-phaser.git](https://github.com/hyandramartins/Sokoban-phaser.git)
 
 *cd Sokoban-phaser/sokoban*

 ## Instala as dependencias

npm install
npm install phaser
npm install vite

## Servidor

npm run dev
